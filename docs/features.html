MB OS Features

Explore the capabilities of our lightweight operating system

Custom Bootloader

Our dual-stage bootloader provides complete control from the first sector, with FAT12 filesystem support and memory detection before loading the kernel.

  • 512-byte stage1 bootloader
  • FAT12 filesystem navigation
  • Memory detection via INT 0x15
  • Protected mode switching

Minimal Kernel

The 32-bit protected mode kernel provides essential services while maintaining a small footprint.

  • Basic memory management
  • Interrupt handling (IDT)
  • Hardware abstraction
  • Text-mode console
  • Basic device drivers

Hardware Support

MyOS includes drivers for essential PC hardware components.

  • 8259 PIC controller
  • 8253/8254 PIT
  • PS/2 keyboard
  • VGA text mode
  • Serial port (COM1)

Development Features

Designed with developers in mind, MyOS provides tools for system programming.

  • Assembly and C development
  • Debugging via serial port
  • Comprehensive documentation
  • Build system with Makefiles
  • QEMU and Bochs support

Technical Specifications

Architecture

  • CPU: x86 (32-bit protected mode)
  • Memory: 1MB+ required
  • Boot: BIOS bootloader
  • Filesystem: FAT12

Development

  • Languages: NASM, C
  • Toolchain: GCC cross-compiler
  • Emulators: QEMU, Bochs
  • Build: GNU Make

Requirements

  • Host OS: Linux, macOS, WSL
  • RAM: 64MB+ recommended
  • Storage: 1.44MB floppy image
  • CPU: i386 compatible

MB OS

A lightweight x86 operating system developed for educational purposes and low-level experimentation.
