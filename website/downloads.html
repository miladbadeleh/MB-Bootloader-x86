System Requirements

  • CPU: i386 compatible or newer
  • RAM: 64MB minimum
  • Storage: 1.44MB floppy drive or image
  • Emulator: QEMU or Bochs recommended

Installation

  1. Download the floppy image
  2. Run in emulator: qemu-system-i386 -fda myos.flp
  3. Or write to USB: dd if=myos.flp of=/dev/sdX bs=512

Build From Source

1

Clone Repository

git clone https://github.com/username/myos.git
cd myos
2

Install Dependencies

# On Ubuntu/Debian
sudo apt-get install nasm qemu make
3

Build System

make all
4

Run in Emulator

make run
